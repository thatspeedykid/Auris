# Auris Driver

This folder contains the FxSound Virtual Audio Device (VAD) driver files,
bundled with the Auris installer.

## Files (not tracked in git — copy locally from FxSound installation)
- `fxvad.sys` — kernel driver
- `fxvad.inf` — driver manifest  
- `fxvadntamd64.cat` — Microsoft signature
- `fxdevcon64.exe` — device installer
- `DfxSetupDrv.exe` — FxSound driver setup utility
